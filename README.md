Open Source Rest API Testing, an alternative software such as Postman, Insomnia, Soap UI, etc

![Show Case](http://supanadit.com/wp-content/uploads/2019/12/restsuite.png)

## Requirements
- Java 11
- Maven

## How to Build Jar

```shell script
mvn clean compile assembly:single
```
